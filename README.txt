The Python Code nlm.py implements the NLM algorithm in Python.
The MATLAB Code used as reference was available at 
http://fr.mathworks.com/matlabcentral/fileexchange/
13176-non-local-means-filter

The NLM function in Python has not been used. 
We implemented the logic so that it can be modified for OBNLM.



The file obnlm.py implements the OBNLM algorithm for speckle noise in Python. 

For OBNLM The Mex file is available at
https://sites.google.com/site/pierrickcoupe/softwares/
denoising-for-medical-imaging/speckle-reduction

The m file is not available.
